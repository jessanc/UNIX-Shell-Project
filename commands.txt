bison -dy filename.y
lex filename.l
gcc lex.yy.c y.tab.c
./a.out 


// windows

For Compiling Lex file only:

	flex hello.l
	gcc lex.yy.c

For Compiling Lex & Yacc file both:

	flex hello.l // lex hello.l
	bison -dy hello.y // yacc -dy hello.y
	gcc lex.yy.c y.tab.c

For Executing the Program

	a.exe
